Introduction

This project analyzes how discounts impact sales, profitability, and customer behavior in an e-commerce store. It also examines factors like social media engagement, ad interactions, and shipping preferences to uncover trends.

Objective

  • Identify the most profitable product category.
  • Analyze the impact of discounts on sales and profitability.
  • Examine customer purchase trends and behavior.
  • Evaluate the influence of social media and advertisements.

Methodology

  • Data Collection: Customer transactions dataset from an e-commerce store.
  • Data Cleaning: Handling duplicates, missing values, and standardizing formats using SQL & Excel.
  • Data Analysis: SQL queries for insights, Tableau for visualization.

SQL Analysis

Finding the Most Profitable Product Category

This query calculates total profit by subtracting cost price from purchase amounts, sorting the results to identify the most profitable category.


SELECT purchase_category, 
       SUM(purchase_amount - cost_price) AS total_profit
FROM ecommerce_consumer
GROUP BY purchase_category
ORDER BY total_profit DESC
LIMIT 1;

Impact of Discounts on Sales

Analyzing total sales with and without discounts to assess whether discounts drive higher revenue.


SELECT purchase_category, 
       SUM(CASE WHEN discount_used = TRUE THEN purchase_amount ELSE 0 END) AS sales_with_discount,
       SUM(CASE WHEN discount_used = FALSE THEN purchase_amount ELSE 0 END) AS sales_without_discount
FROM ecommerce_consumer
GROUP BY purchase_category
ORDER BY sales_without_discount DESC;

Results

  • Electronics is the most profitable category.
  • Discounts increase sales but lower profit margins.
  • High-income customers spend more per purchase.
  • Social media engagement significantly impacts purchasing behavior.

Summary

This project provides insights into e-commerce sales, profit trends, and discount strategies. The results help optimize marketing and sales strategies for higher profitability.
